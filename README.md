# AgenticAI – Smart Last-Minute Doubt Resolver

## 🚀 Overview
AgenticAI is an AI-powered study assistant designed to help students during last-minute exam preparation. It quickly answers doubts by understanding content from notes, PDFs, and past question papers.

## ❗ Problem
During exams, students often run out of time to revise properly. Searching through notes or slides takes too long, and many doubts remain unanswered.

## 💡 Solution
This project provides a smart system where students can ask questions and get instant answers based on their own study materials, saving time and improving understanding.

## ⚙️ Features
- Instant doubt clarification  
- Works with notes, PDFs, and past papers  
- Context-aware answers  
- Saves time during revision  

## 🛠️ Tech Stack
- Python  
- NLP / AI models  
- Document processing  

## 🎯 Goal
To make last-minute revision faster, easier, and more effective for students.
